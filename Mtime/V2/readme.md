# 概要

爬取时光网信息，根据当前显示的内容，完成数据爬取

## 爬虫分层
- 数据获取层
- 数据处理层
- 数据存储层


## 主要方法
```python
main()
实现爬虫的主要功能，主函数
```
```python
Util()
实现外部变量的导入，主要记录数据库的信息

```
```python
getinfo()
获取影片详情页面内容

```
```python
builder()
将爬虫信息内容解析为sql语句

```
```python
connectionDB()
连接数据库并创建链接数据库的实例

```
```python
selector()
爬虫内容检索层，主要用于数据维护更新

```
